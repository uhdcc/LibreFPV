LibreFPV Quickstart guide  
  
Download and run the installer for the latest version of Visual Studio  
In the "Workloads" tab, check the option for "Game development with C++"  
Install  
<img src="https://github.com/user-attachments/assets/64d0f09c-93ad-4748-9926-bfb3bc88b314" width="720">
  
Download Unreal Engine 4.27.2 from Epic Games Launcher  
    Click on “Unreal Engine”, on the left side  
    Click “Library” at the top  
    Click the plus sign next to “Engine Versions”  
    Pick 4.27.2 (if you only see 4.27.0, wait a minute)  
<img src="https://github.com/user-attachments/assets/46dbdc32-3de6-44b9-a4a1-87dcf3b09d06" width="720">
<img src="https://github.com/user-attachments/assets/9c1d6c6d-6ffc-458a-b04f-872d129e1cd9" width="720">
  
Accept license agreement  
Click "Options" and unselect everything.  
Click install  
<img src="https://github.com/user-attachments/assets/f57112e9-3ea1-4dfc-8454-1917755e76d6" width="720">
  
After the engine installs, click "Launch" and install whatever prerequisites it needs  
Click "Fix Now" if you get this popup  
<img src="https://github.com/user-attachments/assets/5f843446-e530-4671-8893-177c0c56372b" width="720">
  
Go to https://github.com/uhdcc/LibreFPV  
Click the green "Code" button  
Click "Download ZIP"  
<img src="https://github.com/user-attachments/assets/2135734d-3ed2-4317-b577-5e2c8a3b8476" width="720">
  
  
Extract the zip file somewhere    
Right click "LibreFPV.uproject" and pick "Generate Visual Studio project files"  
<img src="https://github.com/user-attachments/assets/89410723-7223-45bb-9016-7c63e7dde4f2" width="720">
  
Open "LibreFPV.sln"  
Use "Microsoft Visual Studio Version Selector" if you get this popup  
When it asks you to sign in, click "Skip this for now" and click "Start Visual Studio"  
When the solution opens, from the menu at the top, select Debug -> Start Without Debugging  
Once it's done building, the LibreFPV will open, and you can now launch the project using  
	the "LibreFPV.uproject" file without opening Visual Studio  
<img src="https://github.com/user-attachments/assets/c1f0360e-7703-4573-9e58-c3a983262543" width="720">
